# Robochat
Чат с роботом

## Технологии
- C#
- Avalonia UI
- EF Core + Sqlite

## Запуск
Склонируйте проект
```sh
git clone https://github.com/nickpinecone/robochat.git
```

Запустите
```sh
dotnet run
```
